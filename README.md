# Obsidian + Claude: From Basics to Guru

A structured, hands-on curriculum for mastering Obsidian on Mac and combining it with Claude to become a power user for **data science** and **software engineering**.

You'll go from "what is a vault?" to running advanced PKM workflows with Claude as your thinking partner — generating notes, querying your knowledge base, tracking experiments, and building a true Second Brain.

## How to Use This Repo

1. **Work modules sequentially** — each builds on the last
2. **Do the hands-on exercises** in your actual Obsidian vault (not just read)
3. **Use the knowledge base** as your reference while working
4. **Use `sandbox/`** for free-form practice — paste Claude outputs, test templates, etc.
5. **After each module**, update `sandbox/my-progress.md` with what clicked and what didn't
