# Recommended Plugin Stack

## Core Plugins (built-in, enable in Settings → Core Plugins)

| Plugin | Enable | Purpose |
|--------|--------|---------|
| Daily Notes | ✅ | Daily note per day |
| Templates | ✅ | Basic note templates |
| Canvas | ✅ | Visual whiteboard |
| Bookmarks | ✅ | Pin key notes to sidebar |
| Backlinks | ✅ | See what links here |
| Outgoing Links | ✅ | See where this links |
| Graph View | ✅ | Visual vault map |
| Search | ✅ | Vault-wide search |
| Command Palette | ✅ | Run any command |
| Slash Commands | ✅ | Inline command access |
| Word Count | ✅ | Writing feedback |

## Community Plugins (install via Settings → Community Plugins → Browse)

| Plugin | Priority | Purpose |
|--------|----------|---------|
| **Templater** | Essential | Dynamic templates with JavaScript |
| **Dataview** | Essential | Query vault like a database |
| **Obsidian Git** | Essential | Auto-backup vault to GitHub |
| **QuickAdd** | Essential | Capture + macro automation |
| **Calendar** | High | Calendar for daily notes |
| **Excalidraw** | High | Embedded diagrams |
| **Linter** | High | Auto-format notes |
| **Kanban** | Medium | Task boards |
| **Periodic Notes** | Medium | Weekly/monthly notes |
| **Advanced Tables** | Medium | Better table editing |
| **Natural Language Dates** | Medium | Type "next Monday" → date |
| **Pandoc** | Low | Export to PDF/Word |

## Installation Order

1. Templater
2. Dataview (enable JS queries in settings)
3. Obsidian Git (set up git repo first)
4. QuickAdd
5. Calendar
6. Excalidraw
7. Linter
