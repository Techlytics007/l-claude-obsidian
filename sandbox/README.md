# Sandbox

Free-form practice area. Use this to:

- Test Dataview queries before adding to your vault
- Paste Claude outputs to review before pasting into Obsidian
- Draft templates before finalizing them
- Experiment with Mermaid diagrams
- Try Templater syntax

## Files to Create Here

- `my-progress.md` — your personal notes on each module (what clicked, what confused you)
- `practice-note.md` — a note to experiment with formatting
- `draft-templates/` — templates in progress
